ℹ️ Sobre la Cabina
Los mensajes de WhatsApp en tiempo real requieren conectar la API Oficial de WhatsApp Business (Meta) o servicios como Twilio / 360dialog a un backend propio.

Los datos de oyentes y canciones se obtienen en tiempo real desde tu servidor de streaming (Icecast/Shoutcast) una vez configurada la URL.

La IA Asistente funciona sin backend: analiza los datos disponibles localmente.
Los mensajes aparecerán aquí cuando integres la API de WhatsApp.
